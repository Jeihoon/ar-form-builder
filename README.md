# AR Form Builder

**Version:** 1.0  
**Author:** Amin Rahnama  
**Plugin URI:** https://mypixellab.com

---

AR Form Builder is a powerful and user-friendly WordPress plugin that lets you create fully customized forms using an intuitive drag-and-drop interface—no coding required. Whether you’re building a contact form, registration form, or feedback form, AR Form Builder gives you complete control over layout, fields, and design.

---

## 🚀 Features

- Drag-and-drop form builder (no coding required)
- Create unlimited forms
- Fully customizable fields and layouts
- Responsive and mobile-friendly design
- File upload support
- Google reCAPTCHA integration for spam protection
- Store form submissions in the database
- Clean and lightweight performance
- Works seamlessly with any WordPress theme

---

## ⚙️ Plugin Settings

- **Form Builder Interface** – Create and customize forms visually using drag-and-drop
- **Field Options** – Configure labels, placeholders, required fields, and validation
- **reCAPTCHA Settings** – Enable spam protection using Google reCAPTCHA
- **Database Storage** – Automatically save form submissions for review
- **Form Shortcode** – Easily embed forms anywhere on your site

---

## 📦 Installation

1. Upload the plugin to the `/wp-content/plugins/ar-form-builder` directory.
2. Activate the plugin through the **Plugins** menu in WordPress.
3. Navigate to **AR Form Builder** in the admin dashboard.
4. Create your first form using the drag-and-drop builder.
5. Copy and paste the generated shortcode into any page or post.

---

## 🧼 Why Use AR Form Builder?

Creating forms in WordPress should be simple, flexible, and fast. AR Form Builder removes complexity by providing a clean drag-and-drop interface while still offering powerful features like file uploads, spam protection, and database storage. It’s ideal for developers, designers, and non-technical users alike.
